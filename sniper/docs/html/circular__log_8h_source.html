8// Fast circular log to trace generic events
9//
10// Events consist of a printf-style format string and up to 6 arguments.
11// To reduce overhead, printf itself is not called during logging;
12// instead the format string and all arguments are stored in a circular buffer.
13// On simulation end or abort, the last 1M entries are formatted and written to sim.clog.
14// This means the arguments must remain valid until simulation end, which is fine
15// for integers and const char*. Pointers into dynamically allocated memory will be
16// invalid by the time they are dereferenced and should thus not be used.
