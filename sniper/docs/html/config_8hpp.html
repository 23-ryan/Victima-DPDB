Detailed Description

This file contains most of the interface for the external->in memory configuration management interface. There is a Config class which is the main interface to the outside world for loading files, getting and setting values as well as saving the given configuration. This base class is then derived from for the different given backends. This Config class has a root 'Section'. Each section has a list of subsections and a list of keys. Each key actually contains the value.
