\doxysubsection{Detailed Description}
This file contains the class responsible for \textquotesingle{}flat file\textquotesingle{} configurations. It accepts files very similar to the windows .reg file format, as in sub-\/sections are collapsed like the following\+: \begin{DoxyVerb}---------------------
[main]
value_in_main = 1

[main/subsection]
value_in_sub = "asdf"
"value with spaces" = 4
--------------------- \end{DoxyVerb}


It uses boost\+::regex to perform the parsing. Note that comments are not preserved across saves.
