\doxysubsection{Detailed Description}
\doxyref{Config\+File}{p.}{classconfig_1_1ConfigFile}\+: A flat-\/file interface for the \doxyref{Config}{p.}{classconfig_1_1Config} Class. This file contains the class that is used to interface a flat file for config input / output. It uses boost\+::spirit for the config grammar. 

The grammar is as follows\+: \begin{DoxyVerb}config       =  key* >> section* >> end
section      =  section_name >> key*
key          =  key_name >> '=' >> value
key_name     =  string!
value        =  real | int | string
section_name = '[' >> *lex_escape_ch_p >> ']'
string       = '"' >> *lex_escape_ch_p >> '"' | +(alnum | punct) >> *(alnum | punct ) \end{DoxyVerb}
